# Ensemble Learning Classification

A machine learning project that implements and compares several ensemble learning methods for classification tasks.

## Models
- Majority Voting Classifier
- Bagging Classifier
- AdaBoost Classifier
- XGBoost Classifier

## Features
- Ensemble learning implementation
- ROC AUC evaluation
- ROC curve visualization
- Model performance comparison

## Files
- `ensemble_learning_classification.ipynb` → Jupyter Notebook
- `xgboost_model.pkl` → saved trained XGBoost model

## Technologies
- Python
- Scikit-learn
- XGBoost
- Jupyter Notebook

## Description
This project was created to learn and compare ensemble learning techniques for machine learning classification.
